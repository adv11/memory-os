# Product Requirements

## Vision

MemoryOS is a personal knowledge and learning management system for people learning many subjects at once.

## Target Users

- Software engineers.
- Students.
- Interview preparation candidates.
- Competitive exam aspirants.
- Lifelong learners.

## V1 Scope

- Google login.
- Topic management.
- Learning session management.
- Resource attachment metadata.
- Google Drive file uploads.
- Concept management.
- Knowledge graph visualization.
- Topic dashboard.

## Out of Scope for V1

- Spaced repetition.
- AI-generated questions.
- Semantic linking.
- AI tutor.
- Knowledge decay scores.
- Graph database.
- Vector database.

## Core User Journey

1. User logs in with Google.
2. User creates a topic.
3. User records a learning session.
4. User attaches files or links.
5. User adds important concepts.
6. User views progress and graph.
