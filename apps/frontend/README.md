# Frontend

Target stack:

- Next.js
- TypeScript
- Tailwind CSS
- React Flow

## Implemented

- Next.js App Router scaffold.
- Tailwind CSS setup.
- Login entry linked to backend Google OAuth.
- Browser-side authenticated dashboard loading.
- Basic app shell and Topics route placeholder.

## Run

Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

See `/docs/project/local-development.md` for setup details.
